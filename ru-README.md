# AI VetScan — Ветеринарная AI-диагностическая система, прототип

## О проекте

AI VetScan — это прототип системы поддержки принятия врачебных решений (CDSS), разработанный для ускоренного скрининга патологий дыхательной и сердечно-сосудистой систем у собак и кошек.

Проект объединяет машинное обучение, обработку медицинских данных и ветеринарные базы знаний для поддержки предварительной диагностики.

## Цель проекта

Основная цель — исследовать возможности применения ИИ в ветеринарии для:

* анализа дыхательных звуков;
* классификации сердечных ритмов;
* структурирования ветеринарных данных;
* поддержки принятия диагностических решений.

## Архитектура системы

Система построена как модульный пайплайн:

1. **Data Ingestion**: Использование `data_hunter.py` для сбора и очистки ветеринарных датасетов.
2. **Preprocessing**: `data_mole.py` отвечает за фильтрацию аудио и извлечение признаков для машинного обучения.
3. **Core Analysis**: ML-модели (`PulmoNet`, `CardioNet`) обрабатывают входные данные для классификации состояний здоровья.
4. **Decision Support**: Слои `PharmaDSS` и `VetKnowledge` предоставляют справочный контекст на основе породы и физиологических метрик животного.

## Возможности системы

### 🫁 PulmoScan

Анализ звуков дыхания:

* выявление патологических звуков;
* оценка паттернов дыхания;
* предварительная локализация нарушений.

### ❤️ CardioFlow

Анализ сердечных звуков:

* классификация состояний:
* норма;
* шумы;
* аритмии;
* артефакты.


### 💊 PharmaDSS

База ветеринарных препаратов:

* 323 препарата;
* расчет дозировки по виду животного и весу;
* структурированная информация о препаратах.

### 🧠 VetKnowledge

База ветеринарных знаний:

* 75 пород собак;
* 23 породы кошек;
* генетические риски;
* референсные значения;
* информация о токсичности препаратов.

### 🔍 DataHunter

Инструменты подготовки данных:

* автоматизированный сбор датасетов;
* интеграция с Kaggle CLI;
* мониторинг файлов.

### 🐹 DataMole

Подготовка аудиоданных:

* сортировка;
* валидация;
* предварительная обработка.
